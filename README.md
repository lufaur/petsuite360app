# PetSuite 360 Legal Pages

Legal and support pages for PetSuite 360 iOS app, hosted on GitHub Pages.

## Pages

- **index.html** - Landing page with app info and download link
- **privacy.html** - Privacy Policy
- **terms.html** - Terms of Service
- **support.html** - Support center with FAQ

## Setup

1. Create a new GitHub repository named `petsuite360-legal`
2. Push these files to the `main` branch
3. Go to Settings > Pages > Enable from `main` branch
4. Your site will be live at: `https://<username>.github.io/petsuite360-legal/`

## URLs

After deployment, update the iOS app with these URLs:
- Privacy: `https://<username>.github.io/petsuite360-legal/privacy.html`
- Terms: `https://<username>.github.io/petsuite360-legal/terms.html`
- Support: `https://<username>.github.io/petsuite360-legal/support.html`

## Custom Domain (Optional)

To use a custom domain like `petsuite360.com`:
1. Add a `CNAME` file with your domain
2. Configure DNS at your domain registrar
3. Enable HTTPS in GitHub Pages settings
