Questions frequentes

Comment ajouter un nouvel animal ?

Appuyez sur le bouton + sur le Tableau de bord ou naviguez vers l'onglet Animaux et appuyez sur « Ajouter un animal ». Entrez le nom de votre animal, son espece, sa race et sa date de naissance. Vous pouvez ajouter une photo depuis votre appareil photo ou votre phototeque.

Comment configurer les rappels ?

Allez dans l'onglet Rappels et appuyez sur + pour creer un nouveau rappel. Selectionnez l'animal, choisissez un type d'activite (alimentation, medicament, visite veterinaire) et definissez l'heure et la frequence souhaitees. Assurez-vous que les notifications sont activees dans les Reglages de votre appareil.

Comment fonctionne la synchronisation iCloud ?

Une fois activee, vos donnees d'animaux se synchronisent automatiquement sur tous les appareils Apple connectes au meme identifiant Apple. Activez-la dans Reglages → Synchronisation iCloud. Vos donnees sont chiffrees et stockees dans votre compte iCloud prive.

Comment enregistrer une activite de soin ?

Depuis le Tableau de bord, appuyez sur Enregistrement rapide ou allez sur le profil de votre animal et appuyez sur Ajouter un journal. Selectionnez le type d'activite (repas, promenade, medicament, toilettage), ajoutez des notes si necessaire et enregistrez. Les activites sont automatiquement horodatees.

Qu'est-ce qui est inclus dans Premium ?

Premium debloque les animaux illimites (la version gratuite est limitee a 2), le suivi de sante avance, les horaires de rappels personnalises, les rapports et analyses detailles, les galeries photos etendues et le partage familial. Commencez avec un essai gratuit de 7 jours !

Comment annuler mon abonnement ?

Les abonnements sont geres via Apple : Reglages → [Votre nom] → Abonnements → PetSuite 360 → Annuler l'abonnement. Vous conserverez l'acces jusqu'a la fin de votre periode de facturation.

Comment exporter mes donnees ?

Allez dans Reglages → Donnees et confidentialite → Exporter les donnees. Choisissez votre format et ce que vous souhaitez inclure. L'exportation s'enregistre dans votre application Fichiers ou peut etre partagee directement.

Puis-je partager les soins des animaux avec ma famille ?

Oui ! Avec Premium, activez le Partage familial via iCloud. Tous les membres de votre forfait familial Apple peuvent acceder et mettre a jour les informations des animaux, gardant toute la famille coordonnee.

Comment supprimer mes donnees ?

Glissez vers la gauche sur n'importe quel animal, journal ou rappel pour supprimer des elements individuels. Pour tout supprimer, allez dans Reglages → Donnees et confidentialite → Supprimer toutes les donnees. Cette action est irreversible.

Les notifications ne fonctionnent pas

Verifiez ces parametres : 1) Reglages de l'appareil → PetSuite 360 → Notifications doit etre ACTIVE. 2) Les modes de concentration ne doivent pas bloquer l'application. 3) Dans l'application Reglages → Notifications activees. Si les problemes persistent, essayez de reinstaller.

Encore besoin d'aide ?

Vous ne trouvez pas ce que vous cherchez ? Nous repondons generalement sous 24 a 48 heures.
